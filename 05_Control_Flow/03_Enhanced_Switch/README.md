# Enhanced Switch

This folder contains Java examples demonstrating the use of **Enhanced Switch** expressions in Java.

## Examples Included

- `EnhancedSwitch.java` – Demonstrates the basic enhanced switch syntax using the `->` operator.
- `EnhancedSwitchMethod.java` – Demonstrates using an enhanced switch expression inside a method.
- `EnhancedSwitchWithReturn.java` – Demonstrates returning a value using an enhanced switch expression.

> These examples were implemented while learning Enhanced Switch concepts in the Java Masterclass.
