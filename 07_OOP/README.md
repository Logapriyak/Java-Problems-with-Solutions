# Object-Oriented Programming (OOP)

This folder contains Java examples and challenge solutions demonstrating the fundamental concepts of Object-Oriented Programming (OOP).

## Topics Covered

- Classes and Objects
- Instance Fields
- Static Fields
- Instance Methods
- Static Methods
- Constructors
- Encapsulation
- Inheritance
- Polymorphism
- Composition

> These examples were implemented while learning Object-Oriented Programming (OOP) concepts in the Java Masterclass.
